This guide will help you understand how Relcanonical Monthly Subscriptions work. To begin with, API Values are calculated as the total number of Cells from all your uploaded CSV Files. Each time you upload a new CSV File, an API Endpoint is created. Whether it is Private or Public API, If you use an API Endpoint with i.e. 1,000 values to CREATE an HTML Template, that counts as 1,000 API Requests. Also, when you UPDATE the HTML Template, that counts as another 1,000 API Requests. With that being said, if Relcanonical RENDERs i.e. 100 HTML Pages to your Website Visitors in i.e. May, that counts as 100 API Requests for that month. If you have i.e. 1,000 API Requests left in June, such requests will roll over from July until August, which represents 60 days. Kindly note that API Requests cannot be rolled over on the Small Plan and Small7X Plan. In addition to that, if you cancel your monthly subscriptions, all your unused API Requests will expire at the end of the last paid cycle.
